Web4Dev CM

To get started with the web4dev content manager you need to have the proper tools on your server, as well as the content manager js, and php files.

What you need:

  • A Server
  • LAMP stack
  • jQuery (tested on 3.1.1)
  • Font Awesome
  • Web4Dev CM

download

Web4Dev CM

Is that it?

Not quite, there are a couple files that need to be updated by you.

  1. connect.php
  2. isLogged.php

connect.php

This manages connection to the database, and should be updated with your database details

<?php
$id = $_GET["id"];
$content = $_GET["content"];

$servername = "localhost";
$username = "root";
$password = "";
$dbname = "druplicate";

$conn = new mysqli($servername, $username, $password, $dbname);
?>

isLogged.php

We need to check if the user is logged in, and for that you will need to update this function. It returns true if you have a user logged in with editing capabilites.

<?php
function isLogged(){
    session_start();
    if(isset($_SESSION['username']))
    {
        return 1;
    }
    return 0;
}
?>
